## Environment
- Node v14.16.1

## Running Command
### Development
1. clone this project
```
$ git clone https://github.com/op30132/todo-frontend.git
```
2. install package
```
npm i
```
3. run the project
```
npm start
```
Then open your browser on http://localhost:3000.

### Production
1. build the project
```
npm run build
```
